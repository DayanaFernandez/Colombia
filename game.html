Department...Capital...Colombia!

Ready to have some more fun? Welcome to our game corner! Department...Capital...Colombia is our version of rock, paper, scissors.

Note that the table provided is just for informational purposes so you can see how we came up with our version. Colombia is made up of 32 departments and each department has its own capital. Want to send us feedback? You can do so using the comments box at the bottom of the page. Happy playing!

Careful! This game is case-sensitive and spelling counts. For example, its Colombia, not Columbia!

Ready...set..go show that computer who's boss!
